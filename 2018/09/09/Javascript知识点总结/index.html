JSON


JSON的语法可以表示三种基本类型：简单值、对象、数组；

对象


JSON字符串必须使用双引号；
与JavaScript对象字面量区别：没有声明变量，没有末尾的分号，对象属性必须加双引号；

“age”:21,

}
}

解析与序列化


JSON对象


JSON对象有两个方法：stringify()和parse()分别用于 把js对象序列化为JSON字符串和把JSON字符串解析为原生的js值。
默认情况下，JSON.stringify()输出的JSON字符串不包含任何空格字符或缩进，所有函数和原型成员被有意 忽略，值为undefined的任何属性也都会被跳过。

序列化选项


JSON.stringify()的第二个参数是个过滤器，可以是个数组，也可以是个函数。
如果过滤参数是数组，那么结果将只包含数组中列出的属性；如果第二个参数是个函数，传入的函数接受两个参数，属性名和属性值，根据属性名可以知道应该如何处理要序列化的对象中的属性。
JSON.stringify()的第三个参数用于控制结果中的缩进和空白符，参数是数值表示的是每个级别缩进的空格数。只要传入有效的控制缩进的参数值，结果字符串就会包含换行符。如果缩进参数是个字符串，这个字符串被用作缩进字符。

解析选项


JSON.parse()方法也可以接受第二个参数，该参数是一个函数，称为还原函数。如果还原函数返回 undefined表示要从结果中删除响应的键；如果返回其他值则将该值插入结果中。

instanceof和typeof


instanceof：检测某个值是什么类型的对象。
typeof：检测某个值是什么类型。

Array的常用方法


push():在数组末尾添加，返回添加之后的长度，改变原数组；
pop()：移除数组末端的元素，返回被移除的元素，改变原数组；
shift():移除数组的首个元素，返回被移除的元素，改变数组 ；
unshift()：在数组前端添加元素，返回添加后的长度，改变原数组；
concat():不传递参数时复制当前数组并返回副本，传递数组或值，把数组的每一项或值添加到结果数组，不改变原数组。
slice()：基于当前数组的一或多个项创建新数组。一个参数返回从该参数到结束的所有项，两个参数返回参数之间的数，不包括结束，不改变原数组；
splice():向数组中插入或删除项；删除需要两个参数，要删除的项和要删除的个数；插入或替换需要三个参数，起始位置 ，要删除的项，要插入的项；返回一个数组包含 从原始数组中删除的项；改变原数组 ；
reduce()：迭代数组中的所有项，然后返回一个最终值。
